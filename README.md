# Gate Pass Management System


## Technologies
* Node.js
* Express
* Express Messages, Session, Connect Flash & Validation
* MongoDB & Mongoose
* Pug Templating
* Passport.js Authentication
* BCrypt Hashing
* InstascanJs
* QRcode

### Version
0.0.1


### Installation

Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```
